Puerto Arena

Interactive Installation / Motion Graphics

Concept:

Puerto Arena is an interactive installation in which the user has the opportunity to become a miner and see how his actions has consequences. The idea is make aware about bout the problems that brings illegal mining near the Pacific Ocean. We made a code in processing using the Kinect that tracks the sand's height and assigns a color.

Client: College Project
