Puerto Arena

Interactive Installation

Puerto Arena is an interactive installation in which the user has the opportunity to become a miner and see how his actions has consequences.

The idea is make aware about bout the problems that brings illegal mining near the Pacific Ocean.

Process:

The structure is made of recycled wood. At the top is the kinect, a projector and computer with the code.

Process:

We made a code in processing using the Kinect that tracks the sand's height and assigns a color. The more deep

The green represents the vegetation, the more dig the users go they will find water where they find the gold.
