PasaUnLibro

Ui / Ux

PasaUnLibro is a recommendation system based on the user's preferences. The main idea is share the books that you already readed in specific places around the city and grab another.

We create an interface to make easier the process of knowing the place where is the book you want and that recommends books according to your exchanges.

Process:

We upload a lot of book information in a data base on Excel and created a code in Eclipse that reads this file.
